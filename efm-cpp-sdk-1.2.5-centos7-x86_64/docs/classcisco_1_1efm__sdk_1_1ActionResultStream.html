bool cisco::efm_sdk::ActionResultStream::commit ( )

Only non empty rows will be send. Clears the cache after sending the rows.

◆ get_result_stream()

ActionStreamResult& cisco::efm_sdk::ActionResultStream::get_result_stream ( )

Returns the streams action stream result if the streams type cisco::efm_sdk::ActionResultType::Stream.

The action result can be used to send more rows of data to the peer. The action stream result is not thread safe and should not be manipulated from multiple threads.

Exceptions

◆ get_result_table()

ActionTableResult& cisco::efm_sdk::ActionResultStream::get_result_table ( )

Returns the streams action table result if the streams type cisco::efm_sdk::ActionResultType::Table.

The action result can be used to send more rows of data to the peer. The action stream result is not thread safe and should not be manipulated from multiple threads. It shall be closed by the table handler.

◆ set_close_handler()

void cisco::efm_sdk::ActionResultStream::set_close_handler ( std::function< void(const NodePath &path, const std::error_code &ec)> &&  close_handler)

The close handler will be called if the stream is closed.

The stream can be closed by the peer or by calling the close method. A stream with type cisco::efm_sdk::ActionResultType::Values will be called automatically upon setting the result.

Parameters
close_handlerthe handler to be called when the stream is closed

◆ set_result()

bool cisco::efm_sdk::ActionResultStream::set_result ( UniqueActionResultPtr  action_result)

Sets the result of the stream.

The result will determine the type of the stream. All rows of the result will be sent to the peer. If the type is of cisco::efm_sdk::ActionResultType::Values, the stream will be closed immediately after setting the result.
