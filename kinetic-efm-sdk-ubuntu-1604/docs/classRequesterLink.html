void RequesterLink::connected ( const std::error_code &  ec)
inline

Called every time the link connects to the broker.

Parameters
ecThe error code will be set to an error if the connect failed.

This node will not be available at the first start of the requester. It can be added to the responder via the 'Add Node' action of the /downstream/Responder-Link/nodes node in the dataflow editor. The node named 'subscription' has to be added with a 'Value Type' other than none. After the nodes have been added, a value can be set via the @set action of the /downstream/Responder-Link/nodes/test/subscription node. This value should then be displayed in the requesters output as subscription update.
