Detailed Description

The NodePath describes the path in the node tree to a given node.

A path is composed of path components. Each component is a sub-path, which has to be separated by '/' characters. A valid path would be: /sys/links/dataflow. The last component of the path is the name of the node. In this example it is dataflow. The parent path is /sys/links. The root part is /.

NodePath cisco::efm_sdk::NodePath::rebase ( const NodePath &  rhs) const

Removes the given path from this path if it matches, actually rebasing it to the given path.

Example:

cisco::efm_sdk::NodePath p("/downstream/my-link/first_node");
cisco::efm_sdk::NodePath p1 = p.rebase("/downstream/my-link");
// p1 == "/first_node"

If the path is identical to this path, / is returned. If the given path does not match, an empty path is returned.

Parameters
rhsThe path to rebase this path on.
Returns
The rebased path if the given path matches part of this path, /if the path is identical or an empty path if it doesn't match.
