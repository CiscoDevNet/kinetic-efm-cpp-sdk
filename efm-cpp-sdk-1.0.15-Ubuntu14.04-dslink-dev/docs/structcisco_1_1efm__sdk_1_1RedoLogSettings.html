Detailed Description

The settings for each redo log (persistent queue storage).

For each path a QoS 3 subscription exists on a folder is being created.

uint64_t cisco::efm_sdk::RedoLogSettings::min_available_disk_space_threshold_mb_ {50}

The minimum available disk space threshold (in MB).
