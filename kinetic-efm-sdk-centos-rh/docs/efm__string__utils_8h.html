std::string cisco::efm_sdk::string_escape ( const std::string &  s)

Escapes all non digit or alpha characters to its base16 encoded form.

Can be used to encode node names with illegal characters. Mind, that all usages of the name also have to be encoded, i.e. NodePath etc.
